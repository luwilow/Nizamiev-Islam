# Book Tracker

**Автор:** Иван Иванов

## Описание

Приложение для ведения списка прочитанных книг с возможностью добавления новых записей через удобную форму. Доступна фильтрация по жанру и количеству страниц. Все данные сохраняются в файл `books.json`, что позволяет не терять информацию между запусками.

## Инструкция по запуску

1. Установите Python 3.x.
2. Склонируйте репозиторий.
3. Запустите `main.py`:
   ```bash
   python main.py
   ```
4. Используйте форму для добавления книг.
5. Для фильтрации используйте соответствующие поля и кнопки.
6. Данные сохраняются автоматически при добавлении книги.
7. Для сброса данных удалите файл `books.json`.

## Технологии

- Python 3.x.
- Tkinter (стандартная библиотека для GUI).
- JSON (для хранения данных).
- Git (для контроля версий).

## Пример использования

1. Введите: *«1984»*, *«Джордж Оруэлл»*, *«Антиутопия»*, *«328»*.
2. Нажмите «Добавить книгу».
3. Книга появится в таблице.
4. Введите жанр *«Антиутопия»* в поле фильтра и нажмите «Фильтровать» — отобразится только эта книга.
